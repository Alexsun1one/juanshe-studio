# Skill: 叙事去 AI 腔(prose-humanize)

> **来源/署名**:蒸馏自 [haowjy/creative-writing-skills](https://github.com/haowjy/creative-writing-skills) 的 `llm-writing` guardrail(Apache-2.0,需保留 NOTICE / 署名)与 [blader/humanizer](https://github.com/blader/humanizer)(MIT)。本文件是**方法蒸馏**,非原文转载。

## Purpose
针对**叙事 / 虚构 prose**(不是说明文)删除"一眼 AI"的写作惯性。与 `style/de-ai-tone`(文章 / 资讯向去 AI)互补:那个管观点文的腔,这个管故事的笔。**只改表达,不改情节与事实**。

## Applies To
小说、故事化内容;在写手出稿后、定稿前,作为风格编辑 / 评审的一道质检。

## Input
一段叙事 prose。

## Output
同义改写后的 prose + 命中的 AI 痕迹清单(供作者复核)。

## Rules(命中即改写)
**对话腔泄漏进正文**:"这不是 X,而是 Y"(纠正没人有的误解)、"让我们…""说白了""值得一提的是" → 叙事里删掉,直接写。
**意义注水 / 拔高**:"标志着…的转折""仿佛预示着什么""空气仿佛凝固" → 删,改成具体动作或感官。
**purple prose(过度文采)**:堆叠形容词、把每件事都做成华丽比喻("她的眼眸宛如深海般幽蓝难测") → 砍成一个准确的具体细节。一段里比喻 ≤1 个,且必须服务于此刻的情绪 / 信息。
**telling 命名情绪**:"他感到一阵恐惧 / 愤怒 / 释然" → 换成身体信号或动作。
**抽象代替具体**:"环境很压抑""气氛很紧张" → 给能看见的东西(墙皮、声音、温度)。
**句式同构**:连续相同主语 / 相同句式开头、段落等长 → 打散,长短交替。
**转折词滥用**:然而 / 与此同时 / 尽管如此 频繁出现 → 用情节 / 动作 / 视角自然转,而非连词硬转。
**强行三连**:rule-of-three 凑排比 → 按真实数量写。
**对话工整**:角色都说完整工整句、轮流播报信息 → 加打断、停顿、答非所问、潜台词。
**装饰**:少破折号 `——`、少 emoji;中文优先逗号句号与「」直引号。

## Style Tokens
节奏:短句 ↔ 长句交替;允许真实的、不完美的口吻;一句独立成段用于冲击,但不连甩。

## Do
- 朗读测试:每句能自然读出口;像人讲故事,不像 AI 播报。
- 保留情节、人物动机、伏笔的**含义**,只改措辞与质感。

## Don't
- 不为"去 AI"删掉真实情绪或改动情节;不口语化到失去叙事质感。
- 不把比喻删光——删的是**廉价 / 华而不实**的比喻,留准确的那一个。

## Examples
原:"空气仿佛凝固了,一种无形的压力笼罩着房间里的每一个人。"
改:"没人说话。椅子腿在水泥地上刮了一下,声音大得刺耳。"
(示意,非固定句)

## Evaluation
**改前扫描**:对话腔泄漏 / 意义注水 / purple prose / 命名情绪 / 抽象环境 / 句式同构 / 转折词堆叠 / 强行三连 / 对话工整 / 破折号·emoji。
**改后审计**:逐句可朗读?情绪是 showing?比喻精简且准确?句段长短有变化?情节与伏笔含义没动?
