# Skill: 卡兹克式叙事(有见识的普通人认真聊一件事)

> **来源/署名**:改编自 [KKKKhazix/khazix-skills](https://github.com/KKKKhazix/khazix-skills)(`khazix-writer`,MIT License,作者「数字生命卡兹克」)。
> 本文件是**特征蒸馏**而非原文转载;具体范句以原仓库为准。商用前请保留原仓库 MIT 署名。

## Purpose
让 AI 写出"活人感"的科技/产品/现象类长文 —— 读完像"一个有见识的普通人在认真聊一件打动他的事",而不是"AI 在输出信息"。

## Applies To
- 平台:公众号长文、知乎专栏、Newsletter(X/小红书可借用其"开头"与"语言密度")
- 内容类型:产品体验、调查实验、现象解读、工具分享、方法论
- 账号类型:个人 IP / 科技自媒体 / 创作者

## Input
- 一手素材:作者真实做过/用过/踩过的事(进行时,非假设)
- 主线观点(一句话)
- 内容原型(下表选一)

## Output
结构化正文(Content AST),遵循下面的叙事弧与节奏规则。

## Rules
**叙事弧(英雄之旅)**:具体事件切入 → 背景铺垫 → 核心分板块(观点+场景+私人视角+扣主线句) → 文化/认知升维 → 收尾。
**内容原型**:
| 原型 | 核心 | 重心 |
|---|---|---|
| 调查实验 | "我替你去做了这件事" | 过程叙事 + 发现层层递进 |
| 产品体验 | "跟我一起玩" | 场景演示 + 真实感受 |
| 现象解读 | "你注意到了吗?背后是什么?" | 观察→好奇→研究→升维 |
| 工具分享 | "我发现了好东西" | 故事铺垫→展示→结果惊艳 |
| 方法论 | "把压箱底掏给你" | 每节可执行 + 坦诚成本 |

## Style Tokens
- **开头四式**:叙事启动 / 荒诞事实 / 热点破题 / 好奇心驱动;第一句必须激发"然后呢?"。
- **节奏**:长短句交替(避免连续 3 句等长);一句独立成段全文 ≥3 次;现象→表面解释→深层追问→洞察的"层层剥开"。
- **口语化类别**(只学类别,不抄原句):转场过渡 / 表达判断 / 承认不足 / 情绪具象 / 拉近距离 / 标点即情绪。
- **AI 与人的边界**:AI 补证据/类比/背景;人提创意角度、真实经历、核心判断。

## Layout Rules
少用 markdown 小标题与大段加粗;靠段落节奏而非装饰分层(具体平台排版交给 Renderer)。

## Do
- 用真实第一手经历;点名具体工具/产品(不说"某 AI 工具")。
- 大胆"我觉得";先同理对方再给观点。
- 逐一展示(最炸的留最后);埋的钩子后面要响(契诃夫之枪)。

## Don't
- 编造"比如有一次…"假设例子;讲没用过的工具假装体验。
- 教科书开头("在当今 XX 时代""随着 XX 发展");空泛转场("首先…其次…最后")。
- 一次性罗列结论而非逐一展示。
