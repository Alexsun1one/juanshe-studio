# Skill: 去 AI 腔(消除 AI 写作痕迹)

> **来源/署名**:规则蒸馏自 [blader/humanizer](https://github.com/blader/humanizer) 与中文版 [op7418/Humanizer-zh](https://github.com/op7418/Humanizer-zh)(均 MIT License)。
> 本文件为**启发式清单**,非原文转载;商用前保留原仓库 MIT 署名。

## Purpose
作为**风格编辑 / 文字编辑 Agent 的最后一道质检**,删除"一眼 AI"的表达,让文本读起来像人写的。**只改表达,不改事实**。

## Applies To
全平台、全内容类型;在 Style Rewriter 之后、发布之前运行。

## Input
一段成品文本(任意平台)。

## Output
同义改写后的文本 + 命中的 AI 痕迹清单(供作者复核)。

## Rules(命中即改写)
**意义注水**:删"标志着重要时刻""充分证明了""折射出更大趋势"等拔高;直接陈述事实。
**系动词回避**:`serves as/boasts/features/充当/堪称` → 直接用"是/有"。
**滥用词**:crucial/pivotal/landscape/interplay/intricate;中文"赋能/抓手/闭环/纵深"等套话 → 换具体说法。
**否定排比**:"不只是 X,更是 Y" / "не仅…而且"模板化 → 拆成直白句。
**强行三连**:rule-of-three 凑数 → 按真实数量写。
**标点/装饰**:少破折号 `——`、少 emoji 装饰、少行内"标题:"、用直引号;中文优先逗号句号与「」。
**含糊归因**:"专家认为/业内人士/相关报道" → 点名具体人/机构/时间。
**过度对冲**:"可能或许大概可以认为" → 给明确判断或明说不确定。
**机器人腔**:"希望对你有帮助/当然可以/让我们深入了解/首先来看看" → 删。

## Style Tokens
节奏:短句↔长句交替,避免等长;允许第一人称真实反应("我注意到""这点我也没想明白")。

## Do
- 用"是/有"等常用词直说;点名来源(人/机构/年份);保留真实观点与情绪。
- 朗读测试:每句能自然读出口。

## Don't
- 不动事实、数据、引用的**含义**(只改措辞)。
- 不为"去 AI"而引入错误信息或过度口语化到失专业。

## Examples
原:"这一举措充分彰显了其在行业版图中的关键地位。" → 改:"这让它在该领域排到了前面。"(示意,非固定)

## Evaluation
**改前扫描**:系动词替身 / 滥用词 / -ing 注水 / 含糊归因 / 机器人签名 / 虚假拔高 / 强行三连 / 破折号·emoji·标题式表头。
**改后审计**:逐句可朗读?段落长短有变化?来源具体?语气一致?该有的观点在?机器人痕迹清零?
