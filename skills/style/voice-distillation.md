# Skill: 作家声音蒸馏(把样本抽象成可解释 Style tokens)

> **来源/署名**:方法参考 [haowjy/creative-writing-skills](https://github.com/haowjy/creative-writing-skills) 的 `style-creator` 思路(Apache-2.0)。本文件是**方法蒸馏**,非原文转载。
>
> **版权红线(硬)**:本技能用于**学习风格特征**,**不复制具体在世博主 / 作家**。样本一律抽象成可解释的 Style tokens;**禁止**大段复制、洗稿、内置某作者原句、产出"某作者仿写器"。

## Purpose
把用户给的写作样本(或一种风格描述)蒸馏成**可版本化、可解释、可复用的 Style Skill**——一组明确的风格 token,而不是一个模仿具体某人的黑箱。

## Applies To
风格编辑(style-rewriter)/ 风格创建(style-creator)角色;新建账号画像或风格库时。

## Input
1-3 段样本文本(用户拥有或授权,或公开通用风格的描述)。

## Output
一个结构化 Style token 表(下方维度),可直接注入风格编辑的系统提示。

## Rules(蒸馏维度——逐项填,不空泛)
- **开头方式**:怎么进场(动作 / 反常事实 / 提问 / 场景)。
- **句子节奏**:平均句长、长短句配比、独立成段的频率。
- **段落策略**:段落长度、分层靠节奏还是靠小标题。
- **语言密度**:信息密度高 / 低,修饰多 / 少。
- **比喻与意象**:用不用比喻、来源域(生活化 / 专业 / 自然)。
- **情绪强度与表达**:外放还是克制,直白还是潜台词。
- **论证 / 叙事结构**:线性 / 层层剥开 / 先结论后展开。
- **互动方式**:跟读者的距离(亲密 / 疏离),提问 / CTA 习惯。
- **标点与排版习惯**:破折号、省略号、引号、emoji。
- **禁忌**:这个风格**不做**什么(明确反例)。
- **适用平台 / 内容类型**。

## Do
- 每个 token 给**可执行描述**(如"句子以短句为主,平均 12-18 字,每 3-4 句插一个独立成段的短句做停顿"),不要写"风格独特"这种空话。
- 抽象到**模式**层面:学"怎么做",不存"原句"。
- 标注样本来源与授权状态。

## Don't
- 不内置 / 输出具体作者的原句或可识别片段。
- 不产出针对某在世博主的仿写器;不鼓励洗稿。
- 不写无法执行的形容词堆砌。

## Examples
样本特征 → token:"开头常用一句荒诞事实勾人;句子短促、口语;每节末尾一句独立成段的判断句收束。"
(模式描述,非原句)

## Evaluation
每个维度都填了且可执行?有明确禁忌反例?**全程没有出现可识别的某作者原句**?产出的是"通用风格"而非"某人仿写器"?
