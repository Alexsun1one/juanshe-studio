## 题材禁忌

- 历史细节失真(明清乱穿、官职混用、礼仪不符朝代)
- 主角靠'我知道历史'万能预知(真实穿越者只知大事件,不知具体细节)
- 现代知识万能(写诗就能名扬天下 / 简单火药就横扫战场)
- 把古人当傻子(主角随便忽悠,所有人都信服)
- 改革无阻力(动了既得利益却没人反对)
- 用'忽然想起后世某段历史'强行推进剧情
- 风格混入修真腔或都市装逼腔,失去历史厚重感

## 历史细节规则

- 朝代背景必须扎实:官制、军制、税制、礼仪、服饰、饮食、货币 全部对应
- 历史人物出场前要有真实生平研究,不能完全 OOC
- 用古人当时的语言习惯说话(称谓、尊卑、文白程度),不要现代化
- 重大历史事件(战役、政变、灾荒)要遵循真实时间线,改写要付出代价
- 主角现代知识应用要有'转译'过程:火药需要找硝、硫、炭并提纯,不是直接给配方
- 经济 / 民生 / 农业 的改良要符合当时生产力(良种、水利、税制改革,而非工业)

## 权谋与政治

- 朝廷势力网状分布:皇帝 + 太子 + 皇子 + 后宫 + 文臣 + 武将 + 宦官 + 外戚 + 地方
- 任何决策都涉及多方利益,牵一发动全身
- 改革派 / 保守派 / 投机派 各有立场和现实考量
- 政治斗争不是'好人 vs 坏人',而是利益博弈
- 君臣关系微妙:皇帝既要用又要防,臣子既要忠又要保命
- 反派要有真实动机(权 / 财 / 信念 / 派系),不能脸谱化
- 主角介入大事必须付出代价:暴露身份 / 树敌 / 失去某物

## 叙事指导

第一章必须把朝代、地理、身份、阶层交代清楚,让读者立刻入戏。
日常细节(饮食、起居、出行、社交)是历史质感的根基,每章都要有。
对话要分'朝堂语'(文言)、'士大夫私语'(半文白)、'市井语'(白话方言),不可一锅煮。
战争场面要有兵种、阵法、地形、补给的实际描写,不能只'两军交战 主角一刀胜'。
真正高水准的历史穿越是'让读者感觉自己也活在那个时代',而非'看主角装逼'。
