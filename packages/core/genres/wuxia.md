## 题材禁忌

- 武功体系前后矛盾(内功/外功/轻功的设定不一致)
- 侠客突然失格(为推剧情背叛人设的恩义观)
- 武学突破靠秘籍直给,无修炼过程
- 江湖政治儿戏(门派恩怨说翻就翻,无人情/利益铺垫)
- 历史背景模糊或穿帮(明清服饰用在唐宋)
- 用"绝世武功""天下无敌"跳过具体招式与场景
- 风格混入修真腔(气运/天道)或现代腔(系统/数据)

## 武学规则

- 内功 / 外功 / 心法 / 招式 / 兵器 / 轻功 / 暗器 各自独立,有相生相克
- 突破必有铺垫:打坐、机缘、生死搏杀、名师指点、秘籍参悟
- 高手对决以"招"为最小单位:每招都要有具体动作、攻防、破解
- 武学传承讲门派源流:同门有共同根基,跨派融合是大事
- 兵器有品级与传说,贵重兵器易引发恩怨
- 内伤、走火入魔、毒功 是常用拖延剂,要有现实可信的恢复路径
- "天下第一"不能轻许;高手榜要相对稳定,频繁刷新会贬值

## 江湖伦理与人情

- 侠义、恩仇、信义、报恩 是行为驱动核心(不靠系统、不靠任务)
- 门派 / 帮会 / 朝廷 / 草莽 / 异族 五方势力相互制衡
- 江湖人有江湖人的规矩:不偷不抢、有恩必报、有仇必复,但讲究"以武会友"
- 男女情感讲"知己""红颜""嫁鸡随鸡"等古典模式,不要现代恋爱观
- 人情债与道义抉择构成主要内在冲突
- 不要让主角随意杀人;每次大开杀戒都要付出代价(声誉、仇敌、内疚)

## 叙事指导

武学描写要有节奏感:招式名+动作+破绽+应对+变招,不可流水账。
江湖场景善用酒楼、客栈、镖局、码头、寺观、深山 等地理符号,自然带出势力划分。
人物对话用半文半白的口吻,但不要堆砌"在下""阁下""贵号"等过度礼数,显假。
侠义抉择放在情节高潮点,体现"侠之大者为国为民"的内核。
真正高水准的武侠是"以武写人",武学境界即人物心境。
