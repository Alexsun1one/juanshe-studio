## 题材禁忌

- 男主无理由痴恋女主('因为你很特别',但说不出哪里特别)
- 女主只会卖惨 / 等救;现代女性应该有自己的事业与判断力
- 男主'霸道'写成强迫与跟踪,而非保护与尊重
- 前任 / 反派只会撒泼无理性,缺乏真实动机
- 误会用一句话就能解开,却拖了 20 章
- 商业 / 职场背景空洞(没人真在工作,所有人都在谈恋爱)
- 性别刻板印象(男强女弱、男 fix 女 cry、男主导女配合)

## 现代背景规则

- 都市设定要扎实:具体行业、公司架构、职位层级、薪资水平
- 男女主职业要有真实可信度(CEO 不是 25 岁、医生有规培、律师要执照)
- 家庭背景要有具体细节(父母关系、兄弟姐妹、经济状况、传统观念)
- 朋友圈层有真实功能:同事、闺蜜、发小,各自有性格与作用
- 时代细节真实:微信、外卖、网约车、双 11、996 等都是日常元素
- 婚恋市场的现实压力(房车、彩礼、生育、丁克)可以入戏但要有立场

## 情感推进

- 心动要有具体细节(一个表情 / 一句话 / 一个动作),不是'她突然觉得他很迷人'
- 暧昧期的'忽近忽远' 是经典张力,但不能拖到读者烦
- 误会要有合理触发点,且有合理解开方式(误会引发 → 错位证据 → 关键人物揭破)
- 第三者要有立体动机:不是'坏人想抢',而是'她也有合理理由喜欢'
- 同居 / 旅行 / 吵架 / 和好 是日常爽点,要写出真实的生活感
- 长辈介入要有时代特征(父辈观念 / 家族压力 / 婚嫁议价)

## 叙事指导

第一次相遇必须有记忆点(场景独特 / 反差强烈 / 命运感),避免平庸 meet-cute。
内心戏要有节制:言情不是 90% 内心独白,而是动作 + 对话 + 偶尔内心。
对话要有现代年轻人腔调:不要文绉绉,也不要傻白甜,适度俏皮 + 真诚。
浪漫场景(吻 / 表白 / 求婚)要有铺垫,不能空降;氛围、地点、时机都要精心设计。
事业线要真实推进(女主有自己的工作成就,不是依附男主),HEA 才有说服力。
