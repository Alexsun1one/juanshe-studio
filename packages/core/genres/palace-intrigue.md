## 题材禁忌

- 反派全员脑残(只会下毒 / 推井 / 撕脸,缺乏权谋深度)
- 主角金手指过强(滴血验亲秒破 / 一眼看出毒方)
- 不交代后宫 / 家族等级体系(主子 / 奴婢 / 妃嫔 / 主母 各有规矩)
- 圣宠 / 主母青眼 凭空获得,无铺垫无博弈
- 用'心机''毒辣''阴狠'标签化人物,缺乏具体行为支撑
- 宫廷 / 大族日常生活失真(请安 / 膳食 / 节庆 / 礼仪)
- 把所有人都写成敌人,缺乏盟友 / 中立者的真实立场

## 后宫 / 大族规则

- 等级体系严格:皇后 - 贵妃 - 妃 - 嫔 - 贵人 - 常在 - 答应 各有待遇 / 权限
- 家族架构清晰:嫡 / 庶 / 长 / 幼 / 旁支 各有地位与利益
- 财权 / 人事权 / 子嗣权 是博弈核心,圣眷 / 主母信任是放大器
- 内务管理(膳房 / 库房 / 茶房 / 浣衣局)是隐形战场
- 仆役 / 嬷嬷 / 太监 / 通房 是信息节点,买通她们是常规手段
- 太后 / 太妃 / 老夫人 是终极权威,撇不开

## 斗争与权谋

- 每次正面交锋要有'布局 - 试探 - 出招 - 反招 - 收场' 五步
- 不能正面撕,要借'规矩''礼制''传统''圣意'之名压制对方
- 栽赃 / 离间 / 借刀杀人 是常用手段,要写出合理的因果链
- 盟友 / 中立 / 敌人 关系动态变化,不要二极管
- 子嗣是核心筹码(怀孕 / 保胎 / 流产 / 抚养 都是博弈点)
- 与外朝(皇上 / 父亲 / 兄长 / 大臣)的联动是高阶斗争
- 全身而退比满门荣华更难写,也更高级

## 叙事指导

第一章必须把宫 / 府的等级、人物关系、主角处境交代清楚。
日常细节是斗争伏笔:一杯茶、一道菜、一句问候 都可能是杀招。
对话要藏锋:表面客气 + 内有机锋 + 暗藏陷阱,要让读者读出三层意思。
请安 / 节庆 / 寿宴 是斗法的公开舞台,谁出风头谁就成靶子。
真正高手是'让别人替自己出手' 而非亲自下场,处处留三分。
宫斗不是越狠越好,而是越精越好;狠人多,精明人少。
