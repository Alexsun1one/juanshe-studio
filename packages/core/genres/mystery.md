## 题材禁忌

- 真凶或手法在结局前完全无伏笔,纯属作者强行给出
- 关键线索藏在读者看不到的'内心独白'里,违反 fair play
- 用'灵感''直觉'代替推理过程
- 警察 / 侦探 IQ 时高时低(剧情需要时聪明,需要主角时全员智障)
- 法医 / 刑侦细节失真(死亡时间靠'肤色判断',现代刑侦不用 DNA)
- 受害者只是道具,缺乏社会关系与人物厚度
- 反派动机虚弱('精神病人随机杀人'是最便宜的兜底)

## 推理规则

- Fair play:所有关键线索读者能看到,主角推理出来读者能复盘
- 死亡时间 / 地点 / 手法 必须法医学合理(可以查刑侦教材)
- 询问过程中的微表情、谎言、口误是关键线索
- 至少有 3 个嫌疑人,每人都有动机 + 机会 + 手段,但只有 1 个真凶
- 不在场证明的破解是经典套路,要写得严密
- 物证 / 人证 / 心证 三者相互印证或矛盾
- 真凶最后要'认罪',或被铁证逼到无路可走

## 案件与社会

- 每案要有真实的社会议题(贫富差距、家庭暴力、职场霸凌、教育压迫、医疗腐败 等)
- 受害者不是'符号',要有完整的生活痕迹(职业、家庭、人际、爱好)
- 真凶也是人,有他自己的故事和心理崩溃路径
- 警方 / 法医 / 检方 / 媒体 各有立场和限制
- 主角(侦探 / 警察 / 律师 / 记者)要有自己的私人困境
- 案件解决后要有'余波':受害者家庭、社会影响、主角心理

## 叙事指导

第一章必须有强烈的案发场景,五感描写要到位(尸体细节、气味、温度、光线)。
询问场景要有真实的心理博弈感:对方在隐瞒什么 / 为什么这么说 / 主角在试探什么。
推理章节要给读者跟得上的思路:列举证据 → 排除可能 → 锁定真凶。
不要让真相揭晓得太草率:嫌疑人崩溃 / 哭诉 / 反咬一口 都是经典处理。
社会派推理 vs 本格派推理:前者重动机和社会,后者重诡计和逻辑,要明确风格定位。
