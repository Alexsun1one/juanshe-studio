## 题材禁忌

- 操作描写用'手速快''反应快''天才意识'代替具体动作
- 比赛流程跳跃式('对线 → 团战 → 胜利',中间过程缺失)
- 队友脸谱化(只有'天才上单 / 沙比辅助 / 暴躁打野'三类)
- 比赛靠 BUG / 作弊 / 主角光环胜出
- 战术 / 版本 / 选手转会 等行业元素失真
- 主角全能(LOL / DOTA / CS / 王者 / 吃鸡都顶尖)
- 把比赛写成'装逼大会'缺少团队协作的真实感

## 电竞背景规则

- 选定一款游戏作为核心舞台(LOL / DOTA2 / 王者 / CSGO / OW / 星际 等),其他可作背景
- 游戏机制 / 版本变化 / 英雄强度 / 装备体系 要真实(可参考真实游戏)
- 战队架构清晰:经理 / 教练 / 分析师 / 数据师 / 队员 / 替补
- 联赛体系真实(LPL / LCK / KPL 等;赛季 / 转会窗 / 季前赛 / 季后赛)
- 训练日常:复盘 → 训练赛 → 个人练习 → 心理辅导 / 体能训练
- 选手有职业病(腱鞘炎 / 颈椎 / 失眠 / 焦虑),不是无敌的
- 商业元素(代言 / 直播 / 粉丝 / 黑粉)是真实生态

## 比赛与战术

- 每场比赛要写出'局势'与'转折':前期布局 → 中期博弈 → 后期决战
- 团战不能只写'打输 / 打赢',要写关键操作 + 战术意图 + 失误代价
- 单挑 / 抓人 / 拿龙 / 推塔 都是战术节点,要展示思考过程
- 操作描写要具体(走位 / 技能衔接 / 装备选择 / 时机判断)
- 队友配合不是'我喊他就上',要有真实的指挥 + 默契 + 临场调整
- BP(英雄选择 / 禁用)阶段是博弈核心,要写出双方教练的算计

## 叙事指导

第一章给一场高水准比赛 / 神级操作做钩子,让读者立刻感受到电竞张力。
训练日常不能跳过;它是塑造队伍精神的关键,要写得有温度。
新人加入 → 矛盾 → 磨合 → 成为兄弟 是经典弧,但要写出真实感(各种细节)。
解说视角 / 观众视角 / 选手视角 三种切换,赛事氛围才能层层叠加。
胜利要有重量(付出 / 失败 / 牺牲后的胜利),不是无脑爽。
