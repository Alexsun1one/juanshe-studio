## 题材禁忌

- 朝代背景模糊或穿越(明清服饰 + 唐宋礼仪 + 现代恋爱观)
- 男主从一而终毫无逻辑(三妻四妾的时代,'只爱你一人'要有强动机)
- 女主用现代女权话术撕渣男(可以反抗但要有古代女性视角)
- 用'王爷''将军''丞相'的标签代替人物塑造
- 政治背景空洞(后宫斗 / 家族斗只为推爱情戏)
- 礼节失真(称谓、礼仪、嫁娶流程乱用)
- 用'风度翩翩 / 倾国倾城'代替具体外貌描写

## 古代背景规则

- 朝代设定要明确(架空也要参考一个真实朝代的官制 / 礼仪 / 服饰)
- 男女主身份地位决定行为边界:皇族 / 贵族 / 士族 / 商户 / 平民 / 贱籍 各有规矩
- 婚姻不是单纯爱情:父母之命 / 媒妁之言 / 嫁妆聘礼 / 门当户对 是核心约束
- 称谓系统要正确:皇上 / 殿下 / 大人 / 公子 / 姑娘 / 奴婢 / 妾室,不可乱称
- 古代女性受三从四德约束,叛逆需要付出代价(家族压力 / 名声受损 / 婚配失败)
- 服饰 / 饮食 / 节庆 / 器物 用具体名称(襦裙 / 胭脂 / 元宵 / 团扇)

## 情感张力

- 古代爱情的张力源自'不能':门第不合、政敌之女、亡国遗孤、师徒禁忌、皇命婚约
- 暧昧用克制的肢体语言(指尖触碰、衣袖相挨、并肩看月),不是西方式拥抱
- 诗词 / 书信 / 信物 是古代特有的传情方式
- 一句承诺重于生死(古人重信守诺,现代'我喜欢你'轻飘飘的写法不合时代)
- 三妻四妾 / 政治联姻 是现实压力,男主'退三千佳丽'要有真实代价
- 悲剧美学也是古言传统:不一定要 HEA,凄美收场也是经典

## 叙事指导

第一章必须确立朝代、身份、阶层与初次相遇的场景张力。
对话要带古意:用'妾身''奴家''小生''在下',但别堆砌成酸腐文人腔。
内心戏要用古典意象:'相思如水''心如刀绞''肝肠寸断',而非现代心理描写。
服饰、妆容、举手投足 是塑造人物的重要细节,每章都要有具体描写。
情感推进要克制:一次回眸、一句问候、一个动作 都可能是高潮,而非现代式 'I love you'。
