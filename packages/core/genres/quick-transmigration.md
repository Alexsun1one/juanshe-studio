## 题材禁忌

- 位面之间风格雷同(每个位面都像同一个故事换皮)
- 原主背景交代用'记忆涌入'走流程,缺乏情感深度
- 系统功能万能(任何困境系统都能直接解决)
- 任务标准模糊(只说'攻略 XX 即可',无具体可量化指标)
- 男主同质化(每个位面都是同一种霸总/王爷/总裁)
- 配角无个性(炮灰仇敌只会嘲讽、跪求原谅)
- 把'快'理解成情节潦草,而非节奏紧凑

## 位面与系统规则

- 系统功能要有明确边界:积分商城商品分级、能力使用有冷却 / 限制
- 每个位面有独立的世界规则(修仙 / 末世 / 现代 / 古言 / 科幻),不可混淆
- 任务完成度有量化标准(原主结局逆转 % / 情感线达成 % / 隐藏支线)
- 主角自身能力跨位面有限继承(性格特质 / 主线核心技能可带,装备 / 关系不可带)
- 位面任务有'明线'与'暗线':明线是表面任务,暗线是系统/反派/真相
- 失败有代价:扣分、降级、剥夺记忆,而非简单'重来一次'
- 时间在位面内部正常流动,但在系统/外部可能加速/暂停

## 多位面叙事

- 每位面要有该位面独有的'生活质感':古代有等级礼仪 / 现代有职场情商 / 末世有生存压力
- 原主原身体的'生理记忆'(手势 / 体态 / 口音)与主角灵魂的差异是张力点
- 男主在不同位面是不同人(可以是同一灵魂的转世,也可以是无关角色),要有区分度
- 主线人格逐渐觉醒:从任务工具人 → 在乎位面众生 → 思考系统真相
- 系统人设(冷漠 / 毒舌 / 黑科技 / 神秘),全文风格基调来自系统对话

## 叙事指导

位面入场前 1-2 章必须把原主背景、关键仇敌、任务目标、可用资源讲清楚。
任务过程要让读者看到'主角思考与执行'的全过程,不能跳过推理只展示结果。
打脸场面要有铺垫:对方先嚣张 → 主角让对方进套 → 反转 → 对方破防,层次要分明。
情感戏不能流水账,即便是快穿也要给男女主真正的'认识 - 心动 - 信任 - 承诺' 四步。
跨位面的伏笔回收(同一系统真相 / 同一隐藏反派)是核心爽点,要在前 3-5 个位面铺设。
