## 题材禁忌

- 机甲设定矛盾(同型号能力时强时弱)
- 战术描写用'王牌操作'代替具体战法
- 战场只有主角发挥,队友配角化
- 敌人脸谱化(只会无脑冲锋或自爆)
- 军事术语 / 阶级 / 战略 失真
- 主角无伤碾压,缺乏真实代价
- 把机甲战写成大型 cosplay(炫酷描述 + 一击必杀,缺战术博弈)

## 机甲与军事规则

- 机甲分类清晰(近战 / 远程 / 突袭 / 重装 / 侦察 / 指挥 等)
- 能源 / 弹药 / 装甲 / 速度 / 火力 / 反应 是核心指标,有 trade-off
- 维修 / 补给 / 后勤 是真实约束,机甲不能无限战斗
- 驾驶员选拔 / 训练 / 等级 / 兵种 有完整体系
- 军衔与指挥链严格(列兵 - 班长 - 排长 - 连长 - 营长 - 团长 - 师长 - 司令)
- 战术分级:班 → 排 → 连 → 营 → 团 → 战役 → 战略
- 战略目标决定战术选择(攻坚 / 防御 / 突袭 / 迂回 / 拖延 / 牵制)

## 战斗与战术

- 每场战斗要有'战前情报 → 战术制定 → 临场调整 → 战后复盘' 四阶段
- 单机甲对决要写出操作细节(动作 / 武器切换 / 走位 / 心理博弈)
- 编队战斗要分工(突击 / 火力 / 治疗 / 侦察 / 指挥),呼应真实兵种配合
- 战略层面有大局观(粮草 / 援军 / 民心 / 政治)
- 主角既要技术过硬也要战术头脑;光会打不会指挥,只能是兵不是将
- 死亡是常态:配角 / 战友 / 敌人 / 平民 都可能死,要让读者感受到战争代价
- 胜利不一定有荣耀:有时候是惨胜、孤胆、唯一的幸存者

## 叙事指导

第一章给一场高水准的机甲对决 / 战术博弈做钩子,让读者立刻进入战争氛围。
机甲操作要有'肌肉记忆'感:不是炫耀按了什么键,而是写出驾驶员与机器的合一感。
战场描写要有规模感:数千机甲混战 vs 双方决斗,镜头切换要分明。
军营日常 / 训练 / 战友情 是塑造人物的重要场域,不要全部跳过。
战争的代价(战友死亡 / 主角受伤 / 平民流离 / 城市毁灭)要让读者感受到重量。
真正高水准的军事文是写'人在战争中的选择',而非'装备多帅 / 数据多爆表'。
