## 题材禁忌

- 副本规则交代不清,主角靠'感觉'破解
- 副本场景同质化(都是密室 / 都是丧尸 / 都是古宅)
- 队友脸谱化(只有'萌新 / 老油条 / 反派内奸'三类)
- 主神/系统功能万能,关键时刻必有奖励兑换
- 没有真实死亡风险(角色光环明显,谁都死不了)
- 解密过程跳跃式(读者跟不上思路,主角突然就明白了)
- 风格混入轻松日常或恋爱腔,稀释紧张感

## 副本与积分规则

- 每个副本必须有可被推理的'规则':贴在墙上的提示 / NPC 的话术 / 物件提示
- 副本难度分级(D / C / B / A / S / SS),奖励 + 死亡率成正比
- 积分体系明确:基础奖励 / 隐藏任务奖励 / PK 玩家奖励 / 通关排名奖励
- 主神商店商品分大类:技能 / 物品 / 血脉 / 信息 / 队友复活
- 每副本结束有'结算':通关者名单 + 死亡者名单 + 积分排名 + 隐藏成就
- 玩家间可 PK(暗杠 / 明斗),但要有规则约束(主神禁地 / 副本内可斗)
- 强者归来 / 老玩家梗用于反差,但要在副本逻辑内自洽,不可作弊

## 解密与战术叙事

- 解密要给读者完整线索;主角推理过程要展示(不能只给结论)
- 暗杠要有合理布局:观察队友底牌 → 选时机 → 借副本机制下手
- 团战要分工(肉盾 / 输出 / 控场 / 治疗 / 解密),每位角色都有作用
- BOSS 战不能只拼数值,要利用副本规则 / 场景机制 / 反将 BOSS 一军
- 队友关系动态变化:暂时合作 → 互相提防 → 关键时刻舍命 → 副本结束分道
- 信息差是核心张力:主角知道的 ≠ 队友知道的 ≠ 读者知道的

## 叙事指导

入副本前必须给读者副本的'感官冲击':视觉(场景描写) + 听觉(异响) + 嗅觉(血腥 / 腐烂) + 心理压迫。
副本内每个 NPC、每个物件、每个细节都可能是线索,反复'看到'但不一定'看懂'是套路。
紧张感通过倒计时 + 队友死亡 + 突发事件 维持;不要让读者放松。
暗杠场面写实:不是'我准备杀你了' 而是'下一秒,刀已经在他脖子上'。
主线推进(主神真相 / 老玩家身份)要有前 10-20 个副本的铺垫,一次性揭开会失去神秘感。
