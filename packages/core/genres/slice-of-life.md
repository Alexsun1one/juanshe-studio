## 题材禁忌

- 美化贫困或苦难,把真实的农耕 / 古代生活写成滤镜风
- 主角无所不能(种菜 / 做饭 / 经商 / 育儿 / 医术全会,且都顶尖)
- 反派脸谱化(婆婆 / 妯娌 / 邻居非奸即蠢,只会找茬)
- 时代细节失真(古代用现代调料 / 现代农村用都市装修)
- 把每个日常都写成爽文(打脸 / 报复 / 装逼),失去种田文的温情底色
- 主角与配偶感情线推进过快,缺少日常互动堆积
- 用'幸福生活'四个字代替具体的生活描写

## 生活质感规则

- 时代物价、收入、生活方式必须真实(古代米价 / 90 年代工资 / 现代农村收入)
- 农事 / 家务 / 育儿 都要有具体操作描写(不只是"她做了一桌好菜")
- 食物描写是核心爽点之一:食材、火候、口味、家人反应,层次要全
- 家具 / 衣物 / 节庆 用具体细节(布料、纹样、做法)展示时代质感
- 邻里关系是网状的:不是简单的'好邻居 / 坏邻居',要有立场、利益、性格
- 子女成长是慢线:从出生到成年,每隔 10-20 章一次年龄跨度

## 人物与情感

- 配偶 / 家人 不是道具,要有自己的性格、缺点、成长
- 婆媳 / 妯娌 / 兄嫂 矛盾要写出双方立场,不要单方面'恶毒'
- 感情戏建立在共同生活中:一次默契的眼神 / 一顿合心意的饭 / 一次共渡难关
- 主角的'金手指'(灵泉 / 现代知识 / 厨艺)要克制使用,过度依赖会失温情
- 邻里小事(借米 / 看孩子 / 节庆送礼)是叙事节奏的关键
- 重大节庆(春节 / 中秋 / 婚嫁 / 寿宴)是情感高潮节点

## 叙事指导

每章要有'生活的钩子':一道菜、一件衣、一个节气、一次串门。
对话要有方言味 / 时代腔(古代用半文言、农村用方言、城市用普通话),但别堆砌。
不要让冲突解决得太干净;邻里小别扭、夫妻小拌嘴、子女小叛逆 是日常常态。
描写要慢、要有耐心:写一顿饭可以三页,写一次集市可以五页;这就是种田文的味道。
主线推进可以慢,但不能停;每 10-15 章一定要让生活有可见的改善(房屋 / 田产 / 子女 / 关系)。
